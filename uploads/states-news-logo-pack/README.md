# Website logo pack

Prepared July 23, 2026. All artwork has a transparent background.

## Recommended mapping

| Website entry | Use this file | Note |
| --- | --- | --- |
| LexisNexis | `lexisnexis.svg` | Current vector logo from LexisNexis. |
| Refinitiv | `refinitiv-legacy.svg` | Matches the legacy name currently shown on the website. Refinitiv has retired as a brand; for a fully current presentation, rename the entry **LSEG** and use `lseg-current.svg`. |
| Gale | `gale.svg` | Current vector logo served by Gale/Cengage. |
| Reuters / West | `thomson-reuters-current.svg` | Recommended current parent-company identity. `reuters-current.svg` is included if the relationship was specifically with Reuters News rather than Thomson Reuters/West. |
| Naviga | `naviga-official.png` | Current official color logo. Naviga’s site supplies this artwork as a transparent PNG rather than SVG. At 690 × 169 px it is suitable for display up to about 345 px wide on a 2×/Retina screen. |
| Cision | `cision.svg` | Vector color logo sourced from Cision artwork. |

`preview.png` shows every included option on a light background.
